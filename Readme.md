# Kafka Node Example 

Implementation of https://www.linkedin.com/pulse/introduction-kafka-using-nodejs-pankaj-panigrahi/ tutorial . 

Above tutorial users older version of kafka-node .

## How to run 
To install all the packages
``` node
npm install 
```
To run producer
```
node producer.js
```

To run the consumer 
```
node consumer.js
```
